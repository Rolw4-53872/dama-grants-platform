## التثبيت المحلي

### 1️⃣ استنساخ المشروع
```bash
git clone https://github.com/Rolw4-53872/dama-grants-platform.git
cd dama-grants-platform
```

### 2️⃣ فتح المشروع
```bash
# استخدم VSCode
code .

# أو افتح index.html مباشرة في المتصفح
```

### 3️⃣ تشغيل التطبيق محلياً
```bash
# إذا كان لديك Python مثبت
python -m http.server 8000

# أو استخدم Live Server في VSCode
```

### 4️⃣ الوصول للتطبيق
```
http://localhost:8000
أو
http://localhost:8000/pages/index.html
```

## النشر على GitHub

### 1️⃣ تجهيز التغييرات
```bash
# إضافة جميع الملفات
git add .

# أو إضافة ملفات محددة
git add pages/ js/ api/ *.md
```

### 2️⃣ كتابة رسالة الـ Commit
```bash
git commit -m "feat: إضافة 27 صفحة منفصلة + JavaScript متقدم + API"
```

### 3️⃣ رفع التغييرات
```bash
git push origin main
```

### 4️⃣ التحقق من النشر
```bash
git log --oneline -5
```

## GitHub Pages

### تفعيل GitHub Pages

#### الطريقة 1️⃣: عبر الويب
1. اذهب إلى `Settings` في المشروع
2. اختر `Pages` من القائمة اليسرى
3. اختر `main branch` كمصدر
4. انقر `Save`
5. انتظر دقيقتين للنشر

#### الطريقة 2️⃣: عبر الأوامر
```bash
# لا تحتاج لأي أوامر إضافية، فقط دفع التغييرات
git push origin main
```

## 🐛 استكشاف الأخطاء

### المشكلة: الصفحات لا تحمل
**الحل:**
```bash
# تحقق من امتيازات الملفات
chmod 644 pages/*.html

# تحقق من وجود الملفات
ls -la pages/
```

### المشكلة: البيانات لا تحفظ
**الحل:**
```javascript
// تحقق من console
console.log(localStorage.getItem('damaSystem'));

// امسح التخزين وحاول مجدداً
localStorage.clear();
```

### المشكلة: الأنماط لا تظهر بشكل صحيح
**الحل:**
1. امسح ذاكرة التخزين المؤقت: `Ctrl+Shift+Delete`
2. أعد تحميل الصفحة: `Ctrl+F5`
3. تحقق من وصلات CSS
